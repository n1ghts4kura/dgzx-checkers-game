# 高效开发技巧 — 代码块

为提升开发效率，HBuilderX 将 `uni-app` 常用代码封装成了以 `u` 开头的代码块。

## Tag 代码块

在 template 中输入 `ulist` 回车，自动生成：

```html
<uni-list>
    <uni-list-item title="" note=""></uni-list-item>
    <uni-list-item title="" note=""></uni-list-item>
</uni-list>
```

> 需确保组件已保存在项目的 `components` 目录下。推荐使用 `uni-ui` 项目模板。

### 常用 Tag 代码块

| 代码块 | 说明 |
|--------|------|
| uButton | 按钮组件 |
| uCheckbox | 复选框组件 |
| uGrid | 宫格（需引用 uni ui） |
| uList | 列表（需引用 uni ui） |
| uListMedia | 媒体列表 |
| uRadio | 单选组件 |
| uSwiper | 轮播组件 |
| viewfor | 生成 v-for 循环结构的视图代码块 |
| vbase | 生成基本的 vue 代码结构 |
