# 暗黑模式 (DarkMode) 适配指南

> HBuilder X 3.6.9+ 支持

uni-app 的暗黑模式提供了一批配置和 API，帮助开发者完成自己应用的暗黑模式适配。

## 开启 DarkMode

在 `manifest.json` 中配置：

### app-plus

```json
"app-plus": {
    "darkmode": true,
    "themeLocation": "theme.json"
}
```

### h5

```json
"h5": {
    "darkmode": true,
    "themeLocation": "theme.json"
}
```

### mp-weixin

```json
"mp-weixin": {
    "darkmode": true,
    "themeLocation": "theme.json"
}
```

## 变量配置文件 theme.json

```json
{
    "light": {
        "navBgColor": "#f8f8f8",
        "navTxtStyle": "black"
    },
    "dark": {
        "navBgColor": "#292929",
        "navTxtStyle": "white"
    }
}
```

定义后在 `pages.json` 中以 `@` 开头引用变量：

```json
{
    "globalStyle": {
        "navigationBarBackgroundColor": "@navBgColor",
        "navigationBarTextStyle": "@navTxtStyle"
    }
}
```

## 获取当前主题

若 `manifest.json` 对应平台配置中声明了 `"darkmode": true`，则 `uni.getSystemInfo` 的返回结果会包含 `theme` 属性，值为 `"light"` 或 `"dark"`。

## 监听主题切换事件

通过 `uni.onThemeChange` 监听主题变化，`uni.offThemeChange` 取消监听。

## 页面 CSS 适配

通过 CSS 媒体查询 `prefers-color-scheme` 适配不同主题：

```css
.some-background { background: white; }
.some-text { color: black; }

@media (prefers-color-scheme: dark) {
    .some-background { background: #1b1b1b; }
    .some-text { color: #ffffff; }
}
```

## 完整配置示例

### pages.json

```json
{
    "globalStyle": {
        "navigationBarBackgroundColor": "@navBgColor",
        "navigationBarTextStyle": "@navTxtStyle",
        "backgroundColor": "@bgColor",
        "backgroundTextStyle": "@bgTxtStyle"
    },
    "tabBar": {
        "color": "@tabFontColor",
        "selectedColor": "@tabSelectedColor",
        "backgroundColor": "@tabBgColor",
        "borderStyle": "@tabBorderStyle",
        "list": [
            {
                "iconPath": "@iconPath1",
                "selectedIconPath": "@selectedIconPath1"
            }
        ]
    }
}
```
