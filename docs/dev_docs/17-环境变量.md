# 环境变量配置

## 配置方式一览

| 方式 | 适用场景 |
|------|----------|
| vue-config.js | 修改 webpack 配置，定义环境变量 |
| package.json | 自定义条件编译平台时配置环境变量 |
| .env 文件 | HBuilderX 中 uni-app vue3 / uni-app x (4.25+) 及 CLI 创建的项目 |

## .env 文件

适用于以下项目类型：

- **HBuilderX** 中的 `uni-app vue3` 项目
- **uni-app x (4.25+)** 项目
- **CLI** 创建的项目

在项目根目录中放置 `.env` 文件即可指定环境变量。

### Vue2（CLI 方式）

参考 [Vue CLI 环境变量](https://cli.vuejs.org/zh/guide/mode-and-env.html)

### Vue3（Vite 方式）

参考 [Vite 环境变量与模式](https://cn.vitejs.dev/guide/env-and-mode.html)

## vue-config.js

在 `vue.config.js` 中可修改 webpack 配置，定义环境变量。

## package.json

在自定义条件编译平台时，可在 `package.json` 的 `env` 节点下配置环境变量。

## 注意事项

- uni-app x 不支持自定义环境变量
- `.env` 文件中的环境变量需以 `VITE_` 为前缀（Vite 项目）才能在客户端代码中访问
