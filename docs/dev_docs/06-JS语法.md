# uni-app JS 语法

## 概述

uni-app 的 JavaScript API 由两部分组成：

1. **标准 ECMAScript 的 JS API**
2. **uni 扩展 API**

uni-app 基于 ECMAScript 扩展了 `uni` 对象，且 API 命名与小程序保持兼容。

## 标准 JS 和浏览器 JS 的区别

uni-app 的 JS 代码在不同端运行于不同的引擎中：

- **H5 端**：运行于浏览器中
- **非 H5 端（小程序和 App）**：
  - Android 平台：运行在 **V8 引擎**中
  - iOS 平台：运行在 iOS 自带的 **JavaScriptCore 引擎**中
  - 均不运行在浏览器或 WebView 里

> 非 H5 端不支持 window、document、navigator 等浏览器的 JS API，但支持标准 ECMAScript。

## ES6 支持

uni-app 支持绝大部分 ES6 API，也支持 ES7 的 `await/async`。

### 各端差异

- **Android 端**：App 端使用 V8 引擎（与 Chrome 83 一致），支持的语法与 Android 系统版本无关
- **iOS 端**：使用 JavaScriptCore 引擎，支持的语法与 iOS 系统版本相关

### 注意事项

- 默认情况下**不需要**在微信开发者工具中额外开启 ES6 转换
- 但若使用了微信的 wxml 自定义组件（wxcomponents 目录下），需注意 uni-app 编译器不处理这些文件中的 ES6 代码

## App 端

### Android 平台

- JS 脚本运行在独立 Google **V8 引擎**中，版本与 **Chrome 83** 一致
- vue 页面渲染在系统 WebView 中，受 Android 系统版本影响
- nvue 页面使用系统原生 View 渲染

### iOS 平台

- JS 脚本运行在 iOS 提供的 **JavaScriptCore** 引擎中
- vue 页面渲染在系统 **WKWebView** 中
- nvue 页面使用系统原生 View 渲染
